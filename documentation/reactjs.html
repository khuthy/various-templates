1. OVERVIEW

Crms is a well designed fully responsive admin template that makes the web administration process easier than ever. It is equipped with various functionalities and is platform compatible.

1. 1. Features

  • Creative Design: Crms is a bold theme that organizing content to follow natural eyes, striking balance between beautiful and visual hierarchy.
  • Fully Responsive: The theme passed Mobile-Friendly Test – by Google. We also tested carefully the theme on almost devices: laptop, tablet, and phone. All elements are 100% responsive and ready for all devices.
  • Pixel Perfect Design
  • Color & Fonts is easily changed
  • Customer Reviews, Customer accounts, Compare Products, Grid & List layout, product fillter
  • Newsletter, Social Sharing
  • W3C Validated Code
  • Bootstrap 5.x used
  • Compatible Browsers: IE8+, Firefox, Safari, Opera, Chrome

1. 2. Installation Guide

  • Open the terminal in your root directory & to create the project run the following command:

npx create-react-app Project_name

  • By running the following command, you will be able to get all the dependencies in your node_modules folder:

npm install (or) npm install --legacy-peer-deps

  • To run the project, you need to run following command in the project directory. It will compile the jsx files & all the other project files. If you are making any changes in any of the .jsx file then you need to save the file, it will automatically consider this command and run the entire code.

npm start

  • To install any plugins in your application, you have to run the following command:

npm install plugin_name --save

  • If you want to run the project and make the build in the production mode then run the following command in the root directory, otherwise the project will continue to run in the development mode.

npm run build

CRMS offers you the best framework7 template created by the latest version of HTML and CSS with super responsive mobile template. One of the most important features of a CRMS Mobile template is to Maintain Projects.

Installation Guide

  • After unzipping framework7 Files, you will find our product CRMS Mobile Template->framework7(Source code), Documentationin the following Main Files folder.
  • Open HTML folder, copy all files and paste in your localhost or server root folder. Now you can run this Template in any web browser from your PC or Laptop.

For example:
localhost/crms/index.html or www.example.com/index.html

Requirements

Working with our CRMS product, before going to edit all the HTML, CSS and JS files you will need a text editor.

In this case, we recommended the Sublime Text 3 editor or Notepad++ as it allows you to edit the files efficiently and all the editable files are located in "CRMS->framework7" folder.

Browser Support

How to change Theme Font ?

Go to custom.css and line number 33. Here we have the google font family "Mada", sans-serif

@import url("https://fonts.googleapis.com/css2?family=Mada:wght@400;500;600;700;900&display=swap");

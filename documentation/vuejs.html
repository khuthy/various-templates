1. 2. Installation Guide

  • Open the terminal in your root directory,By running the following command you will get a vue-cli:

npm install -g @vue/cli or yarn global add @vue/cli

  • Open the terminal in your root directory and run the following command for installing plugins used in the application:

npm install

  • Open the terminal in your root directory and run the application as a below command:

npm run dev

  • Now you can run this Template in any web browser from your PC or Laptop.
  • vuejs/template is a folder name, you can change as per your wish how you used the folder

http://localhost:8080/vuejs/template/index
